Usage

The FilterProtonData module converts CSV data into point and fiber (track) data. The properties are used to manipulate and filter this data.
Connections to the PointRenderer, PointConnector and the FiberDisplay are most often used but are not the only possible connections

In the Select Columns category you can select which colum represents which data. E.g. Which column is the x coordinate, which the y...

The Filtering category allows you to filter particles by their particle type. It also allows you to decide whether only primaries or secondaries should be shown.

For the Visualization you can set the size and color of the particles via their deposited energy. It is also possible to change the color gradient.

In the Event Id Limitation category you can change the range of displayed Event Ids.
